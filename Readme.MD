## Webpack konfigurace

Tento repozitář obsahuje nakonfigurovaný webpack pro základní použítí.

### Instalace

Před použítím je nutné spustit instalaci knihoven 

```bash
npm install
```

### Dev server

Pro vývoj aplikace je připraven skript, který po uložení souboru překompiluje data

```bash
npm run dev
```

### Build produkce

Pro produkční použití je zde skript, který vytvoří statický HTML soubor a JS skript se všemi závislostmi.

```bash
npm run build
```
